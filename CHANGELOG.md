# Changelog

All notable changes to this project are documented here.

---

## [1.0.0] — 2025

### Initial public release

**Core skill (`SKILL.md`)**
- Five-mode detection system: Analyze + Rewrite, Generate, Plan, Personal Story, Rewrite
- Non-negotiable rule set enforced on every output
- Humanizer pass: scans for 8 categories of AI tells before delivery
- CTA decision tree: runs automatically on every generation and rewrite
- Personal story protection: never invents, always asks first

**Reference files**
- `brand-profile.md` — fully templated for public use, all example data removed
- `hook-library.md` — 9 pattern-break hook techniques with psychological reasoning
- `post-anatomy.md` — five-section frame, spacing rules, open loop mechanics, length guide
- `cta-library.md` — three-tier CTA system (Soft/Medium/Direct), rotation rules, decision tree, phrasing examples

**Output format (Analyze + Rewrite mode)**
- 12-section structured output
- Virality score: 10 categories, each rated /10 with one-line reasoning
- CTA tier identification on every analyzed post
- Image recommendation with psychological reasoning
- Maximum 3 hashtags, always
